# Advent of Code 2025

This repo contains solutions for all 12 days of advent of code 2025 and a corresponding test runner.
These tests will not work for you locally as you will need to bring your own input and example data from the Advent of Code website.

Some of my solutions are very slow or poorly written, but that's the joy of Christmas! The test runner has a configuration for how many times each test should execute.
I timed all of my solutions, running them 100 times to get a good benchmark (except for my very slow Day9 part2 which I only run once). Some days include a separate "optimized" solution which was completed after my initial attempt.
